# Amplify AI Copilot Submission
Author: Karin Tschudin
Date: June 2026

--------------------------------------------------
1. OVERVIEW
--------------------------------------------------
This submission demonstrates the application of Microsoft Copilot across multiple business scenarios within a pharmaceutical supply chain context.

The work focuses on identifying, analyzing, and mitigating critical supply chain risks impacting production continuity and FDA/GMP compliance.  
All tasks are based on a consistent case involving three key components:
- BIND567 (quality instability)
- COMP042 (traceability gaps)
- COMP043 (supplier non-compliance)

The deliverables show how Copilot can be used to:
- Transform unstructured information into structured outputs
- Support decision-making with data analysis
- Create professional documentation and communication artifacts

4. HOW TO NAVIGATE AND REVIEW THE SUBMISSION
--------------------------------------------------

Recommended review sequence:

1. Start with Task 1 to understand the initial problem and key supply chain risks.
2. Review Task 2 to see how issues are translated into structured meeting outcomes and actions.
3. Proceed to Task 3 to understand the data-driven analysis and risk evaluation.
4. Review Task 4 for an executive-level synthesis of all findings and recommended actions.
5. Refer to the Copilot Prompt Library to understand how each deliverable was created and refined.

Review focus areas:

- Logical flow from problem identification to solution development  
- Clarity and professionalism of outputs  
- Integration of regulatory (FDA/GMP) considerations  
- Effectiveness of Copilot in enhancing productivity and output quality
